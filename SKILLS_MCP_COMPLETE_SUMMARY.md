- `permission-presets.ts` (3 safety presets)

**UI Components**:
- `skills-marketplace.tsx` (beautiful card marketplace)
- `mcp-marketplace.tsx` (matching MCP marketplace)
- `simplified-permissions.tsx` (one-click permissions)
- `install-custom-skill-dialog.tsx` (improved skill installation)
- `install-custom-mcp-dialog.tsx` (improved MCP setup)

**Integration**:
- Updated `extensions-section.tsx` with all new components

### Architecture Highlights

**1. Data-First Design**
- Built-in capabilities as static TypeScript data
- Type-safe interfaces and comprehensive metadata
- Easy to extend and maintain
- Centralized data management

**2. Smart Defaults**
- Safe capabilities enabled automatically
- OS-aware path configuration (Filesystem MCP)
- Clear capability warnings and guidance
- Progressive disclosure (hide advanced options)

**3. Consistent UX**
- Both marketplaces share design language
- Card-based layouts with clear visual hierarchy
- Same search/filter patterns throughout
- Unified color coding (green=skills, blue=MCP, orange=advanced)
