# Team-X User Guide

Welcome to the Team-X user guide. These docs cover everything you need to run your AI organization ‚Äî from hiring your first CEO to asking the Copilot why the frontend team is behind schedule.

New here? Start with **[Getting Started](getting-started.md)**, then browse by phase or jump to whatever you need.

## Keyboard-first primer

Two shortcuts unlock most of the intelligence surface:

- **`Cmd+K` / `Ctrl+K`** ‚Äî open the natural-language command palette. See [Command Palette](command-palette.md) + [Keyboard Shortcuts](keyboard-shortcuts.md).
- **`Cmd+Shift+K` / `Ctrl+Shift+K`** ‚Äî toggle the Copilot sidebar. See [Copilot UI](copilot-ui.md).
