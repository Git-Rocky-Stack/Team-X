The Command Palette (`Ctrl+K` / `Cmd+K`) is your primary interface for natural language commands.

## Customizing Shortcuts

You can customize keyboard shortcuts in **Settings → Keyboard → Customize Shortcuts**.

**Steps:**
1. Open Settings (`Ctrl+,` / `Cmd+,`)
2. Go to **Keyboard** section
3. Click **Customize Shortcuts**
4. Select shortcut to remap
5. Press new key combination
6. Click **Save**

**Reset to defaults:**
- Settings → Keyboard → Reset to Defaults

## Platform-Specific Notes

### Windows
- **Windows key + shortcuts:** Reserved by OS (use Ctrl instead)
- **Alt+Tab:** Switch between applications (Team-X included)
- **Alt+F4:** Close application window

### macOS
- **Cmd key:** Primary modifier (equivalent to Ctrl on Windows)
- **Fn key:** Use with arrow keys for Page Up/Down/Home/End on some keyboards
- **Cmd+Tab:** Switch between applications
- **Cmd+Q:** Quit application (use with care)

### Linux
- **Varies by desktop environment:** Some shortcuts may conflict with system shortcuts
- **Ctrl+Alt+T:** Often opens terminal (may conflict with Team-X)
- **Customize in Settings:** Remap conflicting shortcuts

## Learning Shortcuts

**Best way to learn:**

1. **Press `?` anytime** — Shows context-sensitive help
2. **Explore menus** — Shortcuts listed next to actions
3. **Practice daily** — Muscle memory builds quickly
4. **Customize** — Remap to match your preferences

**Power user tips:**

- **Command Palette is faster than menus** — Use `Ctrl+K` / `Cmd+K` for everything
- **Learn navigation first** — Panel shortcuts (`Ctrl+D`, `Ctrl+T`, etc.)
- **Master filters** — Quickly narrow lists (tickets, employees)
- **Customize for your workflow** — Remap frequently-used actions

## Troubleshooting Shortcuts

**Shortcuts not working?**

1. **Check for conflicts:**
   - Other applications may reserve the same shortcut
   - System shortcuts may override
   - Customize shortcut if conflict exists

2. **Verify keyboard layout:**
   - Non-US layouts may have different key mappings
   - Check Settings → Keyboard → Keyboard Layout

3. **Restart Team-X:**
   - Some shortcuts require restart after changes
   - Quit and relaunch (`Ctrl+Q` / `Cmd+Q`)

4. **Reset to defaults:**
   - Settings → Keyboard → Reset to Defaults
   - Revert customizations if issues persist
