Everything you need to run an AI org

From hiring to shipping — Team-X gives you the full management stack for AI-agent organizations.

55 Curated Roles

Hand-written F10-quality role specs across 6 hierarchy levels. Officers, managers, leads, and ICs — each with unique decision frameworks and communication styles.

Multi-Company Workspace

Run multiple AI organizations side-by-side. Each with its own employees, org chart, goals, projects, and provider settings.

Kanban + Tickets

File tickets, assign them to agents, and watch them work. 4-column drag-and-drop kanban with automatic agent assignment and real-time discussion threads.

One-Click Meetings

Call an all-hands with selected attendees. Interject mid-meeting. Auto-generated minutes with action items that become tickets.

Privacy-First

Zero phone-home. Ever. Runs fully on local models. API keys in the OS keychain. Privacy tier filtering controls which providers agents can use.

10 LLM Providers

Ollama, Anthropic, OpenAI, Google, Groq, OpenRouter, Together, Fireworks, or any OpenAI-compatible endpoint. Adaptive runtime picks the best strategy.

MCP Tool Calling

Agents use Model Context Protocol tools via a singleton host. stdio and SSE connections with tools_allowed/tools_denied enforcement.

Real-Time Telemetry

Company stats, daily usage charts, per-employee breakdown, and cost analysis by provider and model. All data stays local.

File Vault + Backup

SHA256-verified file storage with full-text search. One-click backup/restore of your entire database and vault to a portable archive.

Built for real work

A custom orchestrator — not LangChain, not CrewAI — designed for hierarchical agent organizations with real scheduling, pause semantics, and cost tracking.

Get started in 3 commands

1

Clone and install

$ git clone https://github.com/strategia-x/team-x.git && cd team-x
$ pnpm install
2

Start Ollama (optional — for local models)

$ ollama serve
$ ollama pull llama3.1:8b
3

Launch

$ pnpm dev

The app opens with a seeded company, CEO, and engineer. Chat with your first AI employee.
